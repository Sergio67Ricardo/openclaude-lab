# Instruções do Copilot

## Visão geral

Este documento descreve as instruções operacionais para o Copilot no repositório OpenClaude.

## Organização de instruções

- Priorizar o uso de OpenClaude com Ollama local como padrão.
- Verificar se Cursor Pro está configurado como executor principal.
- Usar MiniMax apenas quando o caminho padrão estiver indisponível.

## Fluxos principais

1. Abrir o ambiente de trabalho em VS Code.
2. Validar o contexto do projeto antes de iniciar o Copilot.
3. Usar comandos de prompt e ferramentas do OpenClaude com foco em tarefas de roteiro e engenharia.

## Quando aplicar um modo de fallback

- Se o servidor Ollama não responder, alternar para MiniMax de forma controlada.
- Documentar a razão da mudança de rota para posterior análise.

## Notas de qualidade

- Assegurar que as instruções sejam claras para novos colaboradores.
- Manter o arquivo atualizado com mudanças de governança de modelos.
